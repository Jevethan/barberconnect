# BarberConnect

A modern marketplace platform connecting customers with verified barbers and barbershops.

## Features

- **Location-based Discovery**: Find barbers near you
- **Verified Profiles**: All barbers are verified for authenticity
- **Authentic Reviews**: Only verified bookings can leave reviews
- **Easy Booking**: Schedule appointments at your convenience
- **For Barbers**: Grow your business and reach more customers

## Tech Stack

- React 19.2.0
- Vite 7.2.2
- Tailwind CSS 4.1.17
- Framer Motion 12.23.24
- React Icons 5.5.0

## Getting Started

1. Install dependencies:
```bash
bun install
```

2. Start the development server:
```bash
bun run dev
```

3. Build for production:
```bash
bun run build
```
